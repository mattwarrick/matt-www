Sphinx doc extensions.

I wrote few extensions for Sphinx documentation.

  • sphinxcontrib-constdata (documatt.gitlab.io/sphinxcontrib-constdata/). Allows showing values, listing tables, and generating links from CSV, JSON and YAML files. Python
  • sphinx-reredirects (documatt.gitlab.io/sphinx-reredirects/). Handles redirects for moved pages. It generates HTML pages with meta refresh redirects to the new page location to prevent 404 errors if you rename or move your documents. Python
